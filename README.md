## Features

- Download videos, playlists, and channels
- Quality selection up to 8K
- Audio-only downloads (MP3, M4A)
- Subtitle downloads with embedding support
- Clean, modern interface
- Cross-platform (Windows, macOS, Linux)

## Installation

### Windows
1. Download `VidGrab Setup vX.X.X.exe`
2. Run the installer
3. Launch VidGrab from Start Menu or Desktop

### macOS
1. Download `VidGrab-vX.X.X-mac.zip`
2. Extract the zip file
3. Move VidGrab to Applications folder
4. On first launch: Right-click â†’ Open (to bypass Gatekeeper)

### Linux
**AppImage (Recommended):**
1. Download `VidGrab-vX.X.X.AppImage`
2. Make executable: `chmod +x VidGrab-*.AppImage`
3. Run: `./VidGrab-*.AppImage`

**DEB (Debian/Ubuntu):**
1. Download `vidgrab_vX.X.X_amd64.deb`
2. Install: `sudo dpkg -i vidgrab_*.deb`
3. Run: `vidgrab`

## Requirements

- Internet connection
- On first launch, VidGrab will download required tools (yt-dlp, ffmpeg) automatically
